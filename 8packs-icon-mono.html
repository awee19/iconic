8packs Icons Monochrome

Description

I custom handcrafted this set of icons for the key features page of the 8packs website when I worked as an interning at the startup company. The icons, which were designed using Abode Photoshop were used to represent each of the key feature offered by the online race management website.

Ideation

The primary colour chosen for the icon was based on the accent colour of the 8packs website (refer to the colour swatch below). The reason for having the icons to be in a monochromatic colour scheme is to complement the base colour of the website accent. The direction for the icons is to be modern and trendy. Thus the icons were designed to have a flat appearance and without texture.
