8packs Icons Coloured

Skills

  • Adobbe Photoshop
  • Adobe Illustrator

Description

This set of icons is a spin-off project from the original monochromatic theme icons which I have created for the 8packs website. It did not take much effort to convert the existing monochromatic icons to their coloured versions. The difficulty comes when I was choosing the colour palette for each of the icon.

Ideation

I wasn't 100% satisfied with the original monochromatic designs as there are some inconsistency in the design. Some of the icons are one-dimensional, while others are 2.5 dimensional. My eventual goal is to convert all of them to 2.5 dimensional while retaining the flat appearance. I kick off this spin-off project by colourising the existing icons. Colouring the icons did not take much effort. The most difficult part of the process came when choosing the individual palette of each icon. To retain consistency across the icons, I have removed the shadow frame from the original design and painted the front frame faint grey.

Specifications

As mentioned earlier in the ideation section, each icon is accompanied by a faint grey (#ececec). Each icon has a custom colour palette. The icon component is horizontal and vertical aligned. Shadow of each of the layer are added to the icons that have multiple layers such as icon_transaction. This is to add depth to the icons.
