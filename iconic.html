Description

The iconic foundry is the home of my design portfolio. I built this website from scratch after I realised that I have to be invited by another user before I could post anything on dribbble.

Ideation

The initial idea for the iconic foundry is to be a place for me to upload all the various collections for icons which I have made (hence its name). However, after I came to realised that I can only post on dribbble if I have an invitation by another dribbbler. I thus changed the entire concept of the website from an icon only portal to a place to showcase all my design collections. The theme of the website is simple and minimal. I have decided to use a neutral greyscale theme for the iconic foundry so that the colour scheme of the website will not clash with the colour schemes of my other design works.

Brand identity

I wanted to create a minimal, but more importantly a memorable logo for the iconic foundry. The initial logo looks identical the that of Facebook. I therefore reworked it to the current version. The new design seamlessly connects the initial "i" with an stroked square, which represents the backing found in square icons. This is to connects the brand to its original idea of being a web portal for icon designs. The reason why I chose to incorporate a square into the brand logo is to pay tribute to the square icons. I always prefer square icons to circular icons as they give off a stronger appearance due to the angular structure. The font used for the logo is Carbin Semibold created by Pablo Impallari.

Specifications

Layout

The iconic foundry uses a single column design to provides the users with a top-to-bottom flow. The long version of the brand logo can be found at the header of the website. There is sidebar menu found on the top right hand corner of the header. The expandable sidebar menu houses the links to the rest of the website and also the link to my current photography portfolio. The main content area can be further subdivided into three grids in the landing page. This is the area where all my design works are found. Other than the main landing page, the rest of the pages found on the website follows a one column layout to reduce confusion in the users.

Icons

I designed a set of single tone icons specially for the iconic foundry. All the icons are designed to be fitted in a square frame to match the brand identity. Some of the icons were not used in the current version of the website.
