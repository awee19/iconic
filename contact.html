8packs Icons Monochrome

Skills

  • Adobbe Photoshop
  • Adobe Illustrator

Description

I handcrafted this set of customised icons for the key features web page of the 8packs website when I worked as an intern at the startup company. The icons, which were designed using Abode Photoshop were used to represent each of the key features offered by the online race management company.

Ideation

The primary colour chosen for the icon was based on the accent colour of the 8packs website (refer to the colour swatch below). The reason for having the icons to be in a monochromatic colour scheme is to complement the colour of the website accent. The direction for the icons is to be modern and trendy. Thus, the icons were designed to spot a texture-less flat appearance.

Colour swatch

Base colour: #F36557

Download Adobe Photoshop colour swatch

Specifications

Each icon is made up of two components; the icon itself and a circular frame. The frame itself can also be broken down into two smaller parts, the front frame, which is transparent, but outlined with the base colour, and a shadow frame which is offset 2% below of the front frame. The shadow frame is 20% darker than the base colour.
